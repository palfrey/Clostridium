# Clostridium

A Befunge-98 interpreter written in Clojure

## Usage

1. `lein run <befunge file>`

or

1. `lein repl`
2. `(use 'clostridium.core)`
3. `(runBefunge "<befunge file>")`

or

`lein test` to run all the pyfunge tests
