# Clostridium

A Befunge-98 interpreter written in Clojure

## Usage

`lein run <befunge file>`

Or `lein repl` then:<br />
`(use 'clostridium.core)`<br />
followed by<br />
`(runBefunge "<befunge file>")`<br />
or<br />
`(runPyfungeTests)`
